# CSS的声明优先级
一个选择器的具体特殊性如下确定：   
1.内联声明的特殊性都是 1，0，0，0   
2.ID选择器的特殊性为 0，1，0，0  
3.类，属性和伪类选择器的特殊性为 0 ，0，1，0   
4.元素和伪元素选择器的特殊性为 0，0，0，1   
5.通配符选择器的特殊性 0，0，0，0   
6.结合符对选择器特殊性没有一点贡献   
7.继承没有特殊性
